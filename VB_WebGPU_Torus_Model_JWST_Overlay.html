Status: The “JWST overlay” is a mock/illustrative layer intended for visual context. It combines a short demo list of high-z entries (e.g., JADES-GS identifiers) with a simplified RA/Dec → local projection for display only. Positions/densities are not a one-to-one rendering of any survey footprint.

Do not use for science: No clustering statistics, number counts, or anisotropy inferences should be drawn from this overlay. It’s for communication and UI testing only.

Planned upgrade: In a future build, we will replace this layer with properly parsed survey catalogs (CEERS/JADES/COSMOS-Web), including RA/Dec/redshift/magnitude handling, and document the projection math and down-sampling strategy.
